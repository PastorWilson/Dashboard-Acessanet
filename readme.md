📡 NOC Dashboard - Monitoramento de Rede

Sistema de monitoramento operacional para NOC (Network Operations Center) desenvolvido para acompanhar em tempo real indicadores da operação de um provedor de internet.

O dashboard centraliza informações de suporte técnico, incidentes de rede, chamados operacionais, status de backbone GPON, comercial, financeiro e almoxarifado, permitindo uma visão rápida da saúde da operação.

A aplicação recebe atualizações em tempo real utilizando WebSocket (Socket.IO), permitindo que a equipe visualize imediatamente eventos como:

aumento de clientes offline

incidentes massivos na rede

chamados críticos

status da infraestrutura

Além disso, o sistema aplica alertas visuais automáticos, destacando situações críticas através de cores e animações para facilitar a identificação rápida de problemas pela equipe NOC.

🚀 Funcionalidades

📊 Dashboard operacional em tempo real

⚡ Atualização automática via Socket.IO

🔴 Detecção automática de massivos de rede

🟡 Identificação de instabilidades

🟢 Status geral da rede

📡 Monitoramento de Backbone GPON

🧑‍💻 Acompanhamento de chamados de suporte (N1, N2, N3)

🧾 Indicadores comerciais e financeiros

📦 Controle operacional de almoxarifado

📺 Interface otimizada para telas grandes e TV (NOC)

🛠 Tecnologias utilizadas

HTML5

CSS3

JavaScript

Socket.IO

Node.js (backend de eventos)

Font Awesome

🎯 Objetivo do projeto

Este projeto foi desenvolvido para centralizar informações operacionais de um provedor de internet, facilitando o monitoramento da rede e a tomada de decisão rápida pela equipe técnica.

A solução foi pensada para ambientes NOC, onde é necessário acompanhar múltiplos indicadores em tempo real.
